## Dependencies

* mongoose
* ejs
* express
* nanoid
* dotenv

## Setup

Clone the repo and install the dependencies 

```bash
git clone https://github.com/ishandeveloper/url-shortener.git
cd url-shortener

npm install
```
To run the app, simply write

```bash
npm start
```
